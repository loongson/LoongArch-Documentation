[[pin-multiplexing-configuration-register]]
=== Pin Multiplexing Configuration Register

Offset Address: `0440`-`0443h`

Attribute: R/W

Default value: `FFFF_FFFFh`

Size: `32` bits

This register contains configuration information related to pin multiplexing.
