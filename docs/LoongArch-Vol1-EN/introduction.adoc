== Introduction

The *LoongArch* architecture (LoongArch) is an **I**nstruction **S**et **A**rchitecture (ISA) that has **R**educed **I**nstruction **S**et **C**omputer (RISC) style.
The LoongArch Reference Manual is used to explain the LoongArch specification.
This is the first of three volumes, which describes the basic part of LoongArch.
